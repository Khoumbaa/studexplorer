Comment utiliser la carte ?

La carte vous indique différents points d'intérêts qui pourraient vous être utiles en tant qu'étudiant.

Vous avez accès à des filtres qui vous permettent d'afficher uniquement ce qui vous intéresse.

Faites glisser vos deux doigts sur la carte pour zoomer et déplacer votre position, ou cliquez simplement dessus sur les boutons + et -.
Cliquer sur un point d'intérêt permet d'afficher toutes les informations sur ce dernier.
